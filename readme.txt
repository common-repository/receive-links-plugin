This plugin is designed to integrate a WordPress site with the Receive Links Link Exchange Network with little work on the users part. 

== Description ==

This is a simple plugin to integrate a WordPress site with the Receive Links Link Exchange Network it works with widgets or in the footer.
Created by Receive Links Support staff.

== Installation ==

1. Make sure you have added your site to the the <a href="http://www.receivelinks.com/" target="_blank">Receive Links Network</a> and selected WordPress Plugin as the script langauge. 

2. Install the Wordpress plugin.

Option 1:	Go to your WordPress plugin Panel clcik "Add New" search for "Receive Links". Find Receive Links Plugin click on the Install link on the far right under Actions. Cick the install button on the plugin page.

Option 2: Download it here Go to plugins in your Wordpress admin panel and click "Add New" select upload and browse for the plugin you just downloaded click "Install Now" to install the plugin.

3. Activate the plugin in your Wordpress Admin panel under plugins

4. Once the plugin is activated you should see a widget in the widget section of the Appearance secion of your Admin Panel.
Just drag and drop that in to your side bar and you are ready to go. If you do not have a side bar then the plugin will automaticly include the client code in the footer of the site.

5. Check setup and activate the site on Receive Links Site.
